# Privacy on this documentation site

The VIPP User Manual is designed to work without audience tracking. The project
does **not** configure web analytics, advertising, session replay, marketing
pixels, or a user-account system on this site.

## What happens in your browser

- Search is performed locally in the browser from the site's generated search
  index; the manual does not send search terms to an analytics service.
- The theme and version controls may store small preferences in browser-local
  storage so the selected appearance or manual version can persist.
- The manual uses the reader's system font stack instead of requesting web
  fonts. Other externally referenced theme assets are copied into the built
  site by the MkDocs privacy plugin.

The site does not ask readers to submit scientific images, workflows, or
personal information. If you follow a link to GitHub, image.sc, PyPI, or another
website, that service's privacy terms apply.

## Hosting records

The static site is hosted on GitHub Pages. GitHub may process ordinary service
and request information as the hosting provider under the [GitHub Privacy
Statement](https://docs.github.com/en/site-policy/privacy-policies/github-general-privacy-statement).
The VIPP project does not receive a separate visitor-level analytics feed from
the manual.

## Future analytics policy

No analytics will be added merely because a service is described as
“privacy-friendly.” A later proposal would require a documented purpose,
minimal event and retention design, institutional/privacy review where needed,
and an update to this page before collection begins.

Do not post confidential, identifiable, or ethics-restricted information in a
public support channel. See [report a problem](../troubleshooting/report-a-problem.md)
for safer reporting guidance.
